MYSQL新建数据库和用户账号
日期 2017-05-03 分类 SQL 标签 #SQL#

登陆数据库：

mysql -h数据库HOST -P端口 -u 管理员账户 -p

然后按提示输入密码即可

显示数据库：

show databases;

创建数据库：

create database 新数据库名;

进入某个数据库：

use 新数据库名;

插入新的用户：

insert into mysql.user(Host,User,Password) values("使用者IP","新用户名",password("密码"));

给用户分配数据库权限：

grant all privileges on 数据库名.* to 新用户名@使用者IP identified by "密码"
